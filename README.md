# Purno Performance v0.1

Mobiele webapp voor de Sharkman Crew.

## In v0.1
- gebruikers Schaiky en Brambo
- dashboard en persoonlijke statistieken
- sessieregistratie met optionele windgegevens
- board, zeil, vin/foil, mast en giek
- triminstellingen en volledige setup-snapshot
- Surfshitt beheren
- eenvoudige set-upanalyse
- lokale opslag, JSON export/import
- installeerbare PWA en offline cache

## Publiceren met GitHub Pages
Upload alle bestanden naar de hoofdmap van de repository. Kies daarna in GitHub:
Settings → Pages → Deploy from a branch → main / root.
